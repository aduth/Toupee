Toupée

Toupée ("ToPay") is an experimental bill tracking application inspired by TodoMVC, a popular learning application used to demonstrate JavaScript MV* frameworks.

The project is a personal exploration of modern JavaScript best practices and design patterns. Specifically, it makes extensive use of RequireJS for modularisation and the Backbone.js MV* framework.
